# Moonsail Website

Static site for moonsailsoftware.com, hosted via GitHub Pages.

## Local Development
To view the site locally, run:
```bash
npx --yes serve .
```
Then visit `http://localhost:3000`.

## Rules
Never push without confirmation first. 

## Adding a new page

When adding a new page, update `sitemap.xml` with the new URL. Then in Google Search Console, use URL Inspect → Request Indexing to get it crawled faster.
