# Repository Guidelines

Canonical repository instructions live in [CLAUDE.md](./CLAUDE.md).

Keep `CLAUDE.md` updated and treat this file as a compatibility pointer for tools that look for `AGENTS.md`.

Critical repository rule: never change the public location or URL path of any app privacy policy page. The apps are already live and depend on their current privacy-policy links working in production. Moving, renaming, or re-slugging any privacy policy path is a breaking production change and must be treated as prohibited unless the user explicitly requests it and has a coordinated rollout plan.
